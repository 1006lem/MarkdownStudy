### :star2: 장단점

| 장/단점|  |
| ---- | ------------------------------------------- |
| 장점 | - 문법이 간결하고 쉽다  |
|      | - HTML로 변환 가능하다  |
|      | - .txt파일로, 별도의 도구 없이 작성 가능하다  |
| 단점 | - 표준이 없다  |
|      | - HTML 마크업 전부를 대신하지는 못한다  |
